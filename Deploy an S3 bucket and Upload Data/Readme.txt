Do the changes needed in serverless.yaml file 
then run following commands

Commands:
Command to install plugin : 
-> npm install --save serverless-s3-sync
here we have installed Serverless S3 Sync plugin : https://www.serverless.com/plugins/serverless-s3-sync

Then command to deploy:
-> sls deploy
